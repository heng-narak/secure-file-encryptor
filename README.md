# Secure File Encryptor

A Python-based tool for secure file encryption and decryption using AES-256-GCM.

## Features
- **AES-256-GCM encryption** for confidentiality and integrity
- **PBKDF2 key derivation** with configurable iterations
- **Secure file headers** containing salt, nonce, and authentication tag
- **Command-line interface** for easy use

### Prerequisites
